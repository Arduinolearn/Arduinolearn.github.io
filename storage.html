EEPROM

We studied a lot from above but when we lost powersupply, we lost every thing stored in arduino.How do we go stroing information even after reset we use EEPROM.EEPROM is a form of solid state non- volatile memory,which is commonly used in systems to stare small amount of data in micro controllers.Its like a quick tiny hardware emmebed on a chip.Were EEPROM devices uses very low level communication protocols such as R2C or one wire which may help easy to integrate into projects.

Typically size of eeprom is from 512bytes to 4096bytes.Storing information in eeprom is simple,specify the address you wish to access and perform the desired function.Some chips may come withthere on build in eeprom,such as Atmega328p.While others dont have can connect external eeprom to chip.

Arduino uno has 1024bytes.This can be acess by using eeprom library.Some thing keep in mind while using eeprom it has a rated life expetency.There is one limitation to take heed of â€“ the number of times we can read or write to the EEPROM. According to the manufacturer Atmel, the EEPROM is good for 100,000 read/write cycles

Use the EEPROM

use the bellow sketches to use the same

#include <EEPROM.h>  // header used for EEPROM

Importand Functions

EEPROM.write(a,b);

The parameter a is the position in the EEPROM to store the integer (0~255) of data b. In this example, we have 1024 bytes of memory storage, so the value of a is between 0 and 1023. To retrieve a piece of data is equally as simple

C = EEPROM.read(a);
Where C is an integer to store the data from the EEPROM position a.

Writing Random number to EEPROM

#include  <EEPROM.h>
int zz;
int EEsize = 1024; // size in bytes of your board's EEPROM
void setup()
{
   randomSeed(analogRead(0));
}
void loop()
{
   for (int i = 0; i < EEsize; i++) //wiriting random values in eeprom
  {
    zz=random(255);
    EEPROM.write(i, zz);
  }
  
}
